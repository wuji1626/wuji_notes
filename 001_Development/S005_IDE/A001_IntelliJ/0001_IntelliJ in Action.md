##1 常用快捷键
1. 删除光标所在行：Ctrl+Y  
2. 代码提示（Content Provider）：Ctrl + Alt + 空格  
Eclipse代码提示Content Assist：Alt + /
3. 文件重命名：Shift + F6
4. 生成Setter、Getter代码：Alt+Insert  
该热键还可以生成构造方法、equals、toString、重写方法、代理方法、Autowired等  
5. 生成main方法：psvm
6. 生成System.out.println()：sout
7. 代码修正：Alt + Enter
